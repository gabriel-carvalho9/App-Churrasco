🥩 Calculadora de Churrasco - Flutter
Projeto inicial desenvolvido em aula para praticar a construção de interfaces e a hierarquia de widgets no Flutter.

🎯 Objetivos da Atividade
Estruturação de Tela: Uso de Scaffold, AppBar e Padding.

Layout: Organização de elementos com Column e SizedBox.

Componentes de Input: Implementação de TextField para captura de dados numéricos.

Interação: Criação de botões (ElevatedButton) para disparar ações.

🛠️ Tecnologias
Dart & Flutter.
