🔥Churras Calculadora
Uma ferramenta prática e estilizada desenvolvida em Flutter para calcular a quantidade ideal de carne, acompanhamentos e bebidas para o seu churrasco.

Funcionalidades
Cálculo Inteligente: Diferencia o consumo médio entre homens e mulheres para evitar desperdícios.

Acompanhamentos e Bebidas: Além da carne, o app projeta a quantidade necessária de guarnições e unidades de bebida.

Interface Premium: Design personalizado com paleta de cores "Charcoal & Orange" e feedback visual moderno.

Validação de Dados: Sistema que impede cálculos com valores vazios ou negativos.

Regra de Negócio
O app utiliza as seguintes métricas base para os cálculos:

Homens: 350g de carne.

Mulheres: 300g de carne.

Acompanhamentos: 30% do peso total da carne.

Bebidas: 3 unidades por pessoa.

Tecnologias e Conceitos Aplicados
Linguagem: Dart

Framework: Flutter

Widgets Utilizados:

StatefulWidget para gerenciamento de estado dinâmico.

TextEditingController para manipulação de inputs.

SingleChildScrollView para garantir responsividade em telas menores.

InputDecoration personalizado para campos de texto.
